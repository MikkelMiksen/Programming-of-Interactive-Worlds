## üî© Core Features

- **Dynamic Temperature System**
  - The world temperature changes throughout the day-night cycle.
  - Player temperature is influenced by proximity to heat sources (e.g., furnaces).

- **Resource Management UI**
  - Clean, data-driven panels showing health, food, water, furnace heat, time, and season.
  - Designed with layout groups for flexible scaling and easy modification.

- **Seasonal Progression**
  - Date and time advance naturally through the cycle, with visual and mechanical effects tied to seasons.

- **Dieselpunk Atmosphere**
  - Visuals inspired by oil-era machinery, industrial rust tones, and minimal high-tech influence.
  - Everything feels heavy, tactile, and slightly grimy ‚Äî survival by engineering, not magic.
